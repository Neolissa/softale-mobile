# Legacy Generator Archive

Этот каталог хранит архив генераторного подхода к вариантам ответов.

Статус:
- генератор в runtime отключен;
- текущий источник контента — только ручная редакторская база;
- новые кампании заполняются вручную: вопрос -> 5 вариантов -> реакция -> ветвление -> финалы.

Почему оставили архив:
- чтобы не потерять старую логику как справочный материал;
- чтобы при необходимости можно было сравнить качество/объем старого и нового подхода.

Правило команды:
- генераторный пайплайн не использовать для боевого контента;
- правки контента вносить только через редакторские словари и сценарную библиотеку.
